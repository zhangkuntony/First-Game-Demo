# 《废土家园》

## 末日生存融合游戏设计方案

**塔防 × 资源采集生产 × 主城建造**

版本：v2.0 设计文档

日期：2026/5/19


# 第一章 游戏基础概念

## 1.1 游戏名称与类型

游戏名称：《废土家园》（Wasteland Haven）

游戏类型：3D 塔防 + 资源采集生产 + 主城自由建造

目标平台：PC（Windows/Mac）为主，后续可移植至主机平台


## 1.2 一句话简介

在怪物肆虐的末日世界中，采集资源、生产防御物资、建造你的末日家园——主动挑战关卡获取稀有掉落，用时间和耐心经营出最强的防线。每一张图纸都是新的开始，每一座建筑都是生存的希望。


## 1.3 目标用户群体

核心玩家：18-35岁，喜欢策略塔防与资源管理的混合玩法

次要玩家：沙盒/建造类爱好者，享受按图纸自由编辑主城的乐趣

潜在玩家：末日题材/生存类游戏的受众群体，喜欢"放置+收集+建造"的休闲策略体验


## 1.4 核心体验定位

本游戏的核心体验围绕 **三大玩法模块的闭环循环** 展开：

### 玩法一：塔防（关卡战斗）
- 玩家主动选择开启关卡
- 每关5-12波怪物进攻，逢5为小Boss关、逢10为大Boss关
- 使用防御塔和防御物资抵御入侵，通关后获得资源掉落和图纸奖励
- 核心体验：策略布局 + 实时战斗 + 主动挑战

### 玩法二：模拟经营（资源采集与物资生产）
- 种菜、开矿、伐木等基础资源采集玩法
- 建造生产建筑，将基础资源 + 关卡掉落物加工为防御物资
- 所有采集和生产按照**真实世界时间**进行
- 核心体验：时间管理 + 资源规划 + 放置收益

### 玩法三：主城建造（自由城市编辑）
- 特殊关卡掉落主城设计图纸（含室内和室外部分）
- 点击主城图标进入城市编辑界面
- 按照图纸使用建筑材料自由编辑和装饰主城
- 核心体验：蓝图驱动 + 自由创造 + 个人空间

### 三系统核心循环

```
【资源采集】种菜/开矿/伐木 → 获得基础原材料
       ↓
【物资生产】原材料 + 关卡掉落物 → 生产建筑加工 → 防御塔/弹药/升级材料
       ↓
【塔防关卡】使用防御物资挑战关卡 → 通关获得资源掉落 + 主城图纸
       ↓
【主城建造】消耗建筑材料 → 按图纸编辑主城 → 解锁更多功能/加成
       ↓ （回到资源采集，形成闭环）
```


# 第二章 世界观体系

## 2.1 末日背景设定

公元2047年，一场被称为"大崩坏"的全球性灾难改变了世界。未知的外星物质坠落在地球各处，引发了一系列连锁反应：生态系统崩溃、动植物异变、人类文明分崩离析。幸存者们聚集在零星的避难所中，在怪物的威胁下艰难求生。

你继承了一块位于战略要地的荒废土地和祖父留下的一批旧时代设备。这里资源丰富但危机四伏——周围游荡着变异生物，远处矗立着未知的威胁。你需要在这片土地上建立属于自己的家园，从零开始打造一座能够抵御一切入侵的堡垒。


## 2.2 时间线

| 时间 | 事件 |
| --- | --- |
| 2045年 | 首次发现外星物质，各国秘密研究 |
| 2046年 | 外星物质失控泄漏，局部区域开始出现生物变异 |
| 2047年 | "大崩坏"爆发，全球性灾难降临 |
| 2048-2055 | 混乱时代，文明倒退，幸存者建立小型聚落 |
| 2056年 | 玩家故事开始时间点 |


## 2.3 主要势力阵营

### 2.3.1 新联邦政府（秩序阵营）

由旧时代军方残余力量组建，试图重建中央集权统治。拥有较强的武装力量和技术储备，但内部官僚主义严重。对玩家态度：可能提供任务和支援，但也要求服从和纳税。是高级建筑图纸和科技的主要来源之一。

### 2.3.2 自由商队（中立阵营）

由多个独立贸易车队组成的松散联盟，穿梭于各聚落之间进行交易。玩家可以通过商队出售多余资源或购买紧缺材料。对玩家态度：友好，纯利益导向的商业关系。

### 2.3.3 异端教派（敌对阵营）

崇拜外星物质的狂热组织，认为"大崩坏"是人类进化的契机。他们控制着某些富饶的资源点和古代遗迹，经常发动袭击并召唤变异生物。击败他们的据点可以获得稀有掉落和特殊图纸。对玩家态度：敌对。


## 2.4 主角身份设定

玩家扮演一位继承了祖父遗志的"守护者"。祖父曾是旧时代的工程师和建筑师，在临终前将一块位于战略要地的土地和一份神秘的初始蓝图交给了你。这块土地地下埋藏着丰富的矿物资源，地上有肥沃的可耕种土地，但同时也处于怪物活动频繁的危险区域。你需要在这里采集资源、建造设施、发展出一座真正的末日家园。


## 2.5 叙事主线框架

序章：继承遗产，初到荒野，完成新手教程（首次采集、首个生产建筑、第一关塔防）

前期：建立基础资源采集点（农场/矿场/林场），搭建初级生产线，逐步通过前期关卡

中期：解锁更多生产配方和防御塔类型，获得第一批主城设计图纸，开始建设主城

后期：面对异端教派的全面进攻，揭开"大崩坏"的部分真相

终章：最终Boss战，决定这片区域的命运，完成终极主城的建造


# 第三章 塔防系统设计

## 3.1 防御塔体系

### 3.1.1 防御塔分类

| 塔类型 | 代表建筑 | 攻击特性 | 适用场景 |
| --- | --- | --- | --- |
| 物理塔 | 机枪塔/箭塔 | 高攻速、单体伤害 | 对付快速移动的小型怪物 |
| 爆炸塔 | 火炮塔/火箭塔 | 范围伤害、溅射效果 | 对付密集的怪物群 |
| 能量塔 | 激光塔/电弧塔 | 穿透攻击、持续伤害 | 对付高血量Boss级怪物 |
| 减速塔 | 冷冻塔/粘液塔 | 降低敌人移速和攻速 | 配合其他塔使用，延长输出时间 |
| 辅助塔 | 修复塔/增益塔 | 治疗周围塔或提供Buff | 增强防线持久作战能力 |

> **注意**：所有防御塔不是凭空生成的，需要通过**生产建筑制造**，消耗基础资源和关卡掉落的稀有材料。

### 3.1.2 防御塔升级系统

每座防御塔最多可升级3次（等级1→4），每次升级提升属性并解锁新技能：

| 等级 | 外观变化 | 属性提升 | 解锁能力 | 升级材料需求 |
| --- | --- | --- | --- | --- |
| Lv1 | 基础造型 | 基础攻击力和射程 | 普通攻击 | 基础建材 |
| Lv2 | 增加装甲/装饰 | 攻击力+30%，射程+15% | 被动技能激活 | 基础建材 + 关卡普通掉落物 |
| Lv3 | 明显强化外观 | 攻击力+60%，射程+25% | 主动技能（有冷却） | 精炼材料 + 小Boss掉落物 |
| Lv4 | 终极形态，特效华丽 | 攻击力+100%，射程+40% | 终极技能（强力但长CD） | 合金材料 + 大Boss掉落物 |


## 3.2 怪物系统

### 3.2.1 怪物分类

| 类别 | 代表单位 | 特征 | 应对策略 |
| --- | --- | --- | --- |
| 普通怪 | 变异鼠/丧尸 | 低血量、高数量、速度快 | 用高攻速物理塔集中火力 |
| 精英怪 | 巨型蜘蛛/暴君 | 中等血量、特殊能力 | 配合减速塔，用爆炸塔处理 |
| 飞行怪 | 变异鸟群/浮游体 | 无视地面障碍、走直线 | 必须部署对空塔或全向塔 |
| 小Boss | 腐化巨兽/精英首领 | 高血量、少量特殊技能 | 第5/15/25...关出现 |
| 大Boss | 深渊巨兽/邪神使者 | 超高血量、多阶段复杂技能 | 第10/20/30...关出现 |

### 3.2.2 怪物AI行为模式

路径跟随型：沿预设路径前进，遇到阻挡时尝试绕行或攻击障碍物

智能寻路型：会根据防御薄弱点动态调整路线（高阶关卡启用）

群体协作型：精英怪可以指挥周围的普通怪，形成战术配合

特殊能力型：某些怪物拥有隐身、分裂、复活等特殊机制（Boss专属）


## 3.3 关卡机制（核心）

### 3.3.1 关卡基本规则

游戏采用 **关卡制** 推进，所有关卡由**玩家主动开启**，不存在被动触发：

| 规则项 | 说明 |
| --- | --- |
| 开启方式 | 玩家在关卡列表中选择目标关卡，点击"开始挑战" |
| 波次范围 | 每关设置 **5-12波** 怪物入侵（随关卡难度递增） |
| 小Boss关 | **每第5关**（第5、15、25...关）出现小Boss |
| 大Boss关 | **每第10关**（第10、20、30...关）出现大Boss |
| 重试机制 | 失败可无限重试，不消耗额外资源 |

### 3.3.2 单关流程

```
① 玩家点击"开始挑战" → 进入关卡准备界面
    ↓
② 准备阶段（30秒）：显示本关波次预览和怪物情报
   - 允许调整已放置的防御塔位置
   - 可使用已生产的消耗品（炸弹、修复包等）
    ↓
③ 进攻阶段：怪物按批次生成并向基地推进
   - 每波之间有15秒休息间隙
   - 可进行紧急维修或使用支援道具
    ↓
④ 结算阶段：
   - S/A/B/C评级（基于漏怪数、剩余生命值、用时等）
   - 发放资源掉落奖励
   - 特殊关卡额外掉落图纸或稀有材料
```

### 3.3.3 关卡分类详细设计

#### 普通关卡（第1-4、6-9、11-14...关）

| 属性 | 说明 |
| --- | --- |
| 波次数 | 5-8波 |
| 怪物构成 | 以普通怪+少量精英怪为主 |
| 难度趋势 | 适中递增，引入新怪物类型 |
| 基础掉落 | 基础资源（废金属/木材/矿石/食材）、少量关卡掉落物 |
| 评级奖励 | C~A级 |

#### 小Boss关卡（第5、15、25、35...关）

| 属性 | 说明 |
| --- | --- |
| 波次数 | 7-10波 |
| 怪物构成 | 混合波次 + 最后1-2波出现**小Boss** |
| Boss特点 | 血量约为普通怪的50倍，拥有1-2个特殊技能 |
| 特殊掉落 | 小Boss专属材料、中级图纸碎片、批量基础资源 |
| 评级奖励 | B~S级 |

#### 大Boss关卡（第10、20、30、40...关）

| 属性 | 说明 |
| --- | --- |
| 波次数 | 9-12波 |
| 怪物构成 | 高强度混合波 + **多阶段大Boss战** |
| Boss特点 | 超高血量（普通怪200倍+），3-4个阶段形态转换 |
| **特殊掉落** | **主城设计图纸（室内或室外部分）**、大Boss核心材料、稀有资源 |
| 评级奖励 | A~S级（S级必出图纸） |

### 3.3.4 关卡难度曲线

| 关卡区间 | 定位 | 波次范围 | 特色 |
| --- | --- | --- | --- |
| 第1-4关 | 新手教学期 | 5-6波 | 教学引导，熟悉操作 |
| **第5关** | **首个小Boss关** | 7波 | 首个Boss检验基础能力 |
| 第6-9关 | 成长期 | 6-8波 | 新怪物类型，地形变化 |
| **第10关** | **首个大Boss关** | 10波 | 中期大考，**首张图纸掉落** |
| 第11-14关 | 进阶前期 | 7-9波 | 高强度混合波 |
| **第15关** | **小Boss关II** | 8-10波 | 更强小Boss |
| 第16-19关 | 进阶后期 | 8-10波 | 多路径地图，策略深度 |
| **第20关** | **大Boss关II** | 11-12波 | 极限挑战，**珍贵图纸** |
| 第21关以后 | 专家阶段 | 9-12波 | 全机制混合，最优解测试 |


## 3.4 地图设计原则

路径多样性：每张地图至少有2-3条可能的入侵路径，迫使玩家分散防守

地形互动：利用高地、河流、桥梁等地形元素创造战术机会

建造限制区：标记不可建造区域，增加布局策略深度

视觉标识清晰：怪物路径用地面标记明确提示

关卡主题差异化：不同关卡有不同的环境主题（废墟都市/变异森林/地下矿井/荒漠哨所等），影响可用塔位和战术选择


# 第四章 模拟经营系统设计（资源采集与物资生产）

## 4.1 三层资源架构

本系统的核心逻辑是 **三层资源转化链**：

```
第一层：基础资源（自然采集获得）
    ↓ 加工
第二层：半成品/材料（生产建筑加工获得）
    ↓ 组合（需要第一层 + 关卡掉落物）
第三层：成品（防御塔/弹药/升级组件等）
```

### 4.1.1 第一层：基础资源（采集获得）

| 资源名 | 采集方式 | 采集建筑 | 采集周期（真实时间） | 主要用途 |
| --- | --- | --- | --- | --- |
| **农作物** | 农场种植收获 | 农田（需建造） | 4-8小时/批次 | 食物来源、部分生化产品原料 |
| **木材** | 林地砍伐 | 林场（需建造） | 2-4小时/批次 | 建筑材料、部分塔配件 |
| **矿石** | 矿山开采 | 矿场（需建造） | 6-12小时/批次 | 金属材料来源 |
| **水源** | 水井/集雨 | 水井（需建造） | 持续产出 | 农业灌溉、生产冷却 |
| **废弃零件** | 废墟搜寻 | 搜寻队派遣 | 4-8小时/批次 | 电子元件、机械零件来源 |

> **真实时间说明**：采集计时以现实世界时间为准，离线期间仍会继续计算产出（设有上限）。例如种植后4小时成熟，玩家4小时后回来即可收获。

### 4.1.2 第二层：加工材料（生产建筑制造）

| 材料名 | 所需原料 | 生产设施 | 生产周期（真实时间） | 用途 |
| --- | --- | --- | --- | --- |
| 精炼金属板 | 矿石×5 + 燃料×2 | 熔炉车间 | 2-3小时 | 建造/升级基础材料 |
| 加固木材 | 木材×5 + 树脂×2 | 木工坊 | 1-2小时 | 建筑结构材料 |
| 电子芯片 | 废弃零件×3 + 精炼金属×1 | 电子工坊 | 4-6小时 | 防御塔核心组件 |
| 化学试剂 | 农作物×2 + 水源×3 | 化验室 | 3-5小时 | 弹药/药剂原料 |
| 建筑石材 | 矿石×8 + 粘合剂×2 | 石材厂 | 3-4小时 | 主城建筑材料 |

### 4.1.3 第三层：成品物资（最终产出）

| 成品类型 | 所需材料组合 | 生产设施 | 用途 | 备注 |
| --- | --- | --- | --- | --- |
| **机枪塔** | 精炼金属板×3 + 电子芯片×1 + 螺丝×10 | 武器工厂 | 塔防战斗部署 | 基础防御单位 |
| **弹药补给箱** | 化学试剂×2 + 精炼金属×2 | 弹药车间 | 战斗中补充弹药 | 消耗品 |
| **塔防升级套件** | 电子芯片×2 + **Boss掉落核心×1** | 升级工坊 | 防御塔Lv2→3升级 | 需要关卡掉落物 |
| **建筑材料包** | 建筑石材×5 + 加固木材×5 + 水泥×3 | 建材工厂 | 主城建造/扩展 | 建造系统专用 |
| **修复工具包** | 精炼金属×1 + 木材×2 | 维修站 | 战斗中修理防御塔 | 消耗品 |


## 4.2 资源采集系统详解

### 4.2.1 采集建筑一览

| 采集建筑 | 解锁条件 | 占地大小 | 产量等级 | 可升级次数 | 最高产量 |
| --- | --- | --- | --- | --- | --- |
| 初级农田 | 初始解锁 | 2×2 | 低（每批20单位） | 3次 | 高（每批80单位） |
| 林场 | 完成3关塔防 | 3×2 | 低（每批15单位） | 3次 | 高（每批60单位） |
| 矿场 | 完成5关塔防 | 3×3 | 低（每批10单位） | 3次 | 高（每批40单位） |
| 水井 | 初始解锁 | 1×1 | 持续产出 | 2次 | 提升产出速率 |
| 废墟搜寻站 | 完成7关塔防 | 2×1 | 低（每批5件） | 3次 | 高（每批20件） |

### 4.2.2 采集规则

- 每个采集建筑有**产量上限**（仓库满则停止产出，需及时收取）
- 采集建筑可**升级**提高单次产量和缩短采集周期
- 不同采集建筑之间有**邻接加成**（如农田靠近水井产量+20%）
- 离线期间**最多累积24小时**的产出（防止长时间不上线导致溢出浪费）
- **一键收获**功能：支持批量收取所有成熟资源

- 支持 **连续生产模式**：设置好后自动重复生产
- 生产完成后需手动领取（不会自动存入仓库）

# 第五章 建造系统设计（主城自由编辑）

## 5.1 主城建造概述

### 5.1.1 核心概念

主城是玩家的个人空间和"家园"，采用**蓝图驱动的自由编辑**模式：

- **主城设计图纸**是建造的前提（通过大Boss关卡掉落获得）
- 图纸包含**室外布局规划**和**室内房间划分**两部分
- 玩家在**城市编辑界面**中使用**建筑材料**按照图纸自由建造和装饰
- 主城不仅是展示空间，还能为塔防和生产经营提供**功能性加成**

### 5.1.2 进入方式

玩家在主界面点击 **"主城"图标** 即可进入城市编辑界面：

- 从塔防/采集界面随时可切换进入主城
- 主城界面支持**俯视规划视角**和**第三人称漫游视角**
- 编辑过程中实时保存，退出时自动保存当前状态


## 5.2 主城设计图纸系统

### 5.2.1 图纸获取途径

| 来源 | 说明 |
| --- | --- |

| **S级评价必得** | 大Boss关卡S级评价100%掉落一张图纸 |

| **活动/事件** | 特殊节日或限时活动可能产出特殊风格图纸 |

### 5.2.2 图纸分类

图纸按内容分为两大类，一套完整的主城需要**室外图纸 + 至少一张室内图纸**：

#### 室外图纸（主城地基布局）

定义主城的整体外部框架：

| 图纸名称 | 地基大小 | 解锁关卡 | 风格描述 |
| --- | --- | --- | --- |

| 废都重建方案 | 20×20格 | 第40关 | 旧时代城市复兴风格的终极蓝图 |

#### 室内图纸（房间内部设计）

定义各个房间的内部布局和功能分区：

| 图纸名称 | 房间数量 | 解锁条件 | 包含房间示例 |
| --- | --- | --- | --- |
| 基础住所图 | 3个房间 | 第10关掉落 | 卧室/储藏室/工作台区 |
| 生活区扩建图 | 6个房间 | 第20关掉落 | 增加客厅/厨房/浴室 |

| 终极家园全套 | 15+房间 | 第40关+S级评价 | 全功能区豪华版 |

### 5.2.3 图纸使用规则

- 每张图纸**只能使用一次**（消耗品）
- 室外图纸决定了主城的**最大占地规模**和**地块划分**
- 室内图纸决定了**房间数量、大小和位置**
- 可以叠加多张室内图纸来扩展房间（不能超过室外图纸的上限）
- 图纸上标记了**推荐功能区位置**，但玩家可自由调整

# 第六章 核心循环与系统联动

## 6.1 三系统资源流转闭环

### 核心循环图

```
┌─────────────────────────────────────────────────────┐
│                                                       │
│  【模拟经营】资源采集与物资生产                        │
│    ├── 农田 → 农作物（4-8h真实时间）                   │
│    ├── 林场 → 木材（2-4h真实时间）                     │
│    ├── 矿场 → 矿石（6-12h真实时间）                    │
│    └── 生产建筑 → 加工原材料 → 半成品 → 成品           │
│         （防御塔 / 弹药 / 建筑材料 / 升级套件）        │
│                       ↓                               │
│  【塔防系统】主动挑战关卡                              │
│    ├── 选择关卡 → 部署防御塔（消耗成品）               │
│    ├── 通过5-12波战斗 → 通关结算                      │
│    ├── 普通关：获得基础资源 + 关卡掉落物              │
│    ├── 小Boss关（逢5）：获得Boss材料 + 图纸碎片        │
│    └── 大Boss关（逢10）：获得**主城图纸** + 稀有材料   │
│                       ↓                               │
│  【主城建造】城市编辑                                  │
│    ├── 消耗建筑材料（来自生产系统）                    │
│    ├── 按图纸自由编辑室内外                            │
│    ├── 建造完成后获得功能加成                          │
│    └── 加成反馈回采集和生产系统（效率提升）            │
│                       ↓                               │
│          回到起点 → 更高效的采集 → 循环增强            │
│                                                       │
└─────────────────────────────────────────────────────┘
```

### 流转关系总结

| 流向 | 内容 | 目的 |
| --- | --- | --- |
| 经营 → 塔防 | 防御塔、弹药、升级套件 | 为关卡战斗提供装备 |
| 塔防 → 经营 | 关卡掉落物（Boss材料、稀有资源） | 用于高级生产和升级 |
| 塔防 → 建造 | **主城设计图纸** | 开启/扩展主城建造功能 |
| 经营 → 建造 | 建筑材料包、装饰材料 | 主城建造和装修的材料来源 |
| 建造 → 经营/塔防 | 功能加成（效率提升、备战加成） | 反哺另外两个系统 |


## 6.2 游戏日常流程

### 6.2.1 上线典型操作流程

建议每次上线按以下顺序操作：

```
1.【登录检查】查看离线期间的采集产出（是否有溢出？）
      ↓
2.【收获资源】一键收获所有成熟的农作物/木材/矿石等
      ↓
3.【处理生产】领取已完成的生产订单，安排新一轮生产
      ↓
4.【盘点库存】检查防御塔储备、弹药存量是否充足
      ↓
5.【决定行动】
     ├─ 资源充足？→ 挑战下一关塔防
     ├─ 资源不够？→ 安排更多采集/生产，等待产出
     └─ 有新图纸？→ 进入主城编辑界面建造
      ↓
6.【塔防战斗】（如果选择挑战）
     → 选择目标关卡 → 部署防御塔 → 战斗 → 结算
      ↓
7.【主城编辑】（如果有新图纸或材料）
     → 进入主城 → 建造/升级/装饰 → 保存
      ↓
8.【下线前确认】确保仓库不满、生产队列已安排好
```

### 6.2.2 时间节奏设计

本游戏的核心节奏是 **"短时操作 + 长时等待"** 的异步体验：

| 时长 | 活动 | 频率 |
| --- | --- | --- |
| 5-10分钟 | 收获资源、安排生产、检查状态 | 每天1-3次（利用碎片时间） |
| 15-30分钟 | 挑战一次塔防关卡 | 根据资源情况灵活安排 |
| 30-60分钟 | 主城编辑和装修 | 获得新图纸或有足够材料时 |
| 离线时间 | 资源自动采集/生产（最多累计24h） | 全天候 |

## 8.2 交互流程设计

### 8.2.1 新手引导流程

分为四个核心教学阶段：

**第一阶段 - 资源采集入门（约5分钟）：**
- 引导建造第一个农田
- 种植第一批作物
- 等待成熟并收获（可使用加速道具跳过等待演示）
- 理解真实时间的概念

**第二阶段 - 物资生产入门（约5分钟）：**
- 引导建造第一个熔炉车间
- 将采集的矿石放入生产队列
- 等待生产完成并领取精炼金属
- 理解资源转化的概念

**第三阶段 - 塔防实战（约10分钟）：**
- 引导进入关卡选择界面
- 选择第1关（教学关）
- 学习放置第一座防御塔
- 经历完整的5波战斗流程
- 理解关卡制和主动开启的概念

**第四阶段 - 主城建造（约5分钟）：**
- 引导通过第10关大Boss（简化版教学）
- 获得第一张室外图纸
- 进入主城编辑界面
- 建造第一个建筑
- 理解图纸驱动建造的概念

### 8.2.2 日常操作流程

熟练后的标准流程（每次上线约10-30分钟）：

```
1. 登录 → 查看离线产出提示
2. 一键收获所有采集资源
3. 领取完成的生产订单，补充生产队列
4. 盘点防御物资库存
5. 决定：打关？还是继续攒资源？
   ├─ 打关 → 选关卡 → 布防 → 战斗 → 结算
   └─ 攒资源 → 排生产/排采集 → 下线等产出
6. 如有新材料/图纸 → 进主城编辑一下
7. 确认生产队列安排妥当 → 下线
```


## 8.3 引擎选型建议

| 引擎选项 | 优势 | 劣势 | 适用评估 |
| --- | --- | --- | --- |
| **Unity** | 生态成熟、Asset Store丰富、C#高效、跨平台好、学习资料多 | 高画质需较多优化工作 | **⭐ 强烈推荐** - 本项目的3D场景+UI+复杂逻辑非常适合Unity |
| Unreal Engine | 顶级画质、Blueprint可视化、开源免费 | 学习曲线陡峭、项目较重 | 适合追求顶级画质的后续迭代 |
| Godot | 轻量开源、无版税、GDScript简单 | 生态较小、3D能力有限 | 适合独立开发原型验证 |

**推荐首选 Unity**：考虑到本项目同时需要复杂的3D场景渲染、多系统UI、真实时间后台逻辑、数据持久化等，Unity的成熟生态和丰富插件能大幅加速开发。


## 8.4 技术模块架构

### 8.4.1 核心模块划分

| 模块名称 | 职责 | 核心功能 |
| --- | --- | --- |
| **GameCore** | 核心框架 | 游戏生命周期、全局配置、存档读写、真实时间管理 |
| **ResourceModule** | 资源采集模块 | 采集建筑管理、离线产出计算、收获逻辑、仓库系统 |
| **ProductionModule** | 生产制造模块 | 生产建筑管理、配方系统、队列管理、产出逻辑 |
| **TowerDefenseModule** | 塔防模块 | 关卡管理（5-12波）、塔逻辑、怪物AI（4类+Boss）、战斗结算、掉落系统 |
| **CityBuilderModule** | 主城建造模块 | 图纸系统、地块管理、建筑数据、自由编辑、加成计算 |
| **UISystem** | 界面系统 | HUD、5大子系统UI、交互反馈、动画过渡 |
| **DataSystem** | 数据系统 | 配置表加载（JSON/XML）、数值平衡、本地化 |

### 8.4.2 关键技术要点

| 技术点 | 方案 | 说明 |
| --- | --- | --- |
| **真实时间系统** | 服务器时间校准 + 本地倒计时 | 采集和生产的核心机制，需考虑离线补偿 |
| **离线产出计算** | 上线时根据离线时长 × 产出速率 计算 | 设24小时上限防止溢出 |
| **对象池** | 高频对象（子弹/怪物/特效）复用 | 优化塔防战斗性能 |
| **数据驱动** | 数值配置外部化为JSON表格 | 方便调整平衡性和做A/B测试 |
| **状态机（FSM）** | 管理：主界面/采集界面/生产界面/塔防战斗/主城编辑 | 清晰的状态切换逻辑 |
| **事件总线** | 模块间通信解耦 | 资源变化通知UI更新等 |
| **寻路系统** | NavMesh / A*算法 | 怪物路径 + 室内NPC移动 |
| **存档序列化** | JSON格式 | 支持版本迁移和手动排查问题 |

## C. 风险与应对

| 潜在风险 | 风险描述 | 应对策略 |
| --- | --- | --- |
| 核心循环吸引力不足 | 采集-生产-战斗-建造四环节是否够有趣？ | 早期原型重点验证，每个环节都要独立有趣 |
| 等待时间过长 | 真实时间生产可能导致玩家流失 | 提供合理的加速道具/观看广告加速选项 |
| 关卡内容消耗过快 | 核心玩家可能快速打完所有关卡 | 关卡数量要足够多（至少40+关），且有大难度跨度 |
| 平衡性复杂 | 三系统相互影响，数值平衡难度大 | 数据驱动配置 + 分系统隔离测试 + 大量用户测试 |
| 图纸获取挫败感 | 大Boss关难打，图纸掉落有随机性 | 保证S级必出图纸 + 保底机制（每10次大Boss关保底一张） |
| 主城内容空洞 | 建完就没事做了？ | 持续推出新图纸 + 装饰内容更新 + 功能性加成建筑 |
